# Superellipse

探索数学中的超椭圆曲线，从圆形到矩形的完美过渡。

## 关于超椭圆曲线

超椭圆曲线（Superellipse）是由 Gabriel Lamé 在 1818 年提出的数学曲线，其方程为：

```
|x/a|^n + |y/b|^n = 1
```

其中：

- `a` 和 `b` 是半轴长度
- `n` 是指数，控制曲线的形状

## 超椭圆曲线特性

- ✅ **数学公式** - |x/a|^n + |y/b|^n = 1
- ✅ **形状变化** - 从圆形到矩形的连续过渡
- ✅ **应用领域** - 计算机图形学、建筑设计
- ✅ **美学价值** - 在艺术创作中的广泛应用
- ✅ **参数控制** - 通过指数 n 控制形状
- ✅ **历史意义** - 由 Gabriel Lamé 在 1818 年提出
